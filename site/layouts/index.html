{{/* ── Quick Start ── */}}

Up and running in seconds

Install sew, point it at a context from the registry, and get a fully wired Kubernetes environment on your machine.

The first run pulls container images and may take a few minutes. Once cached, contexts with image preloading configured start in seconds.

Read the docs 
# Install
go install github.com/a-cordier/sew@latest

# Deploy Gravitee APIM from the registry
sew create --from gravitee.io/apim

# Tear down when done
sew delete

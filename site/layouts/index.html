{{/* ── Quick Start ── */}}

Up and running in seconds

Install sew, point it at a context, and get a fully wired Kubernetes environment on your machine.

Read the docs 
# Install
go install github.com/a-cordier/sew@latest

# Deploy Gravitee APIM with MongoDB
sew create --from gravitee.io/apim/aio/mongodb

# Tear down when done
sew delete
